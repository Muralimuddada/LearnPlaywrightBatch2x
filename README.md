# LearnPlaywrightBatch2x

A repository for learning JavaScript fundamentals and Playwright automation testing.

## Contents

- **chapter_01_Basics/** — JavaScript basics
  - `01_Basics.js` — Introduction to `console.log`, `process.platform`, and `process.arch`

## How to Run

```bash
node chapter_01_Basics/01_Basics.js
node chapter_01_Basics/identifier-rules.js
```

## Requirements

- Node.js
